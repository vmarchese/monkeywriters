# MonkeyWriters README 

MonkeyWriters project. 


## To start working on it
```
git clone https://github.com/vmarchese/monkeywriters.git
cd src
npm install
```

Do not add to git the node_modules dir (a .gitignore file has been created anyway)






## Directory Structure
```
--/
  |--src/            (node.js server)
  |--docs/           (documentation)
```
